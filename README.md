# 权限管理后端

#### 介绍
2023山东大学毕业设计 权限管理后端

#### 软件架构
使用MySQL、Redis、Nginx等


#### 启动过程

1.  启动MySQL、Redis
2.  启动Nginx，Nginx具体配置文件在服务器上
3.  启动项目（注意，profile文件在服务器上，需从服务器下载）

#### 其它
项目采用前后端分离架构。
对于前端，采用/{项目名称，如authority}/#/xxx转发至对应文件夹下的前端项目。
对于后端，接口统一采用/api/{项目名称，如authority}/xxx，由Nginx转发至对应端口。
具体参考nginx配置文件。
